Display Property in CSS
Display Property is Very Important in CSS

There are Mainly 4 important Display Property
Example of Block :- div , p , h1 , form etc
Example of Inline :- span , img , a etc

  1. Block
    • Starts with New Line
    • Full Width
    • Can Apply Height and Width
  2. Inline
    • Not Start From New Line
    • Neccessary Width takes
    • Can not Apply Height and Width
  3. Inline-Block
    • Not Start From New Line
    • Neccessary Width Takes
    • Can Apply Height and Width
  4. None
