Different type of Units in CSS

We use length in width,height,margin,padding , font-size

There are two types of units in CSS

  • Absolute

    There are 6 types of Absolute Unit in CSS

    • Cm
    • Mm
    • In(Inch)
    • Px(1Inch = 96px)
    • Pt(1Inch = 72t)
    • Pc(1pc = 12pt)

  • Relative

    There are 5 types of Relative Unit in CSS

    • Em 
      Em is relative to the size of its direct parent
    • Rem 
      Rem is only realtive to root(html tag)size
    • Vh 
      Vh is 1% of view port height
    • Vw 
      Vw is 1% of view port width
    • % 
      % is relative to parent
