There are Manily 3 types of Selector in CSS

  • Element Selectors 
                    Example :- 
                p {
                    color : blue;
                }
  • Id Selector 
                    Example : - 
                //it is used for style particular Element
                #para 
                {
                    color : red;
                }
  • Class Selector 
                    //it is used to style  group of elements
                .para 
                {
                    color: yellow;
                }
  • Priority of Selector if we use all three selectors in one element then priority sequence - >
    1. ID Selector
    2. Class Selector
    3. Element Selector
