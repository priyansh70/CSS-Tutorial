Grid in CSS

Both Flexbox and css grid has its own importants and should be used for designing a web page

Grid works for 2 dimensional alignment

Flexbox is good for space distribution

some important definition

  1. Grid Column :- The vertical lines of gird items are called columns
  2. Grid Row :- The horizontal lines of gird items are called rows
  3. Grid Gap :- The Space Between each row and column is called Gap
  4. Column Lines :- The lines between columns are called column lines
  5. Row Lines :- The lines between rows are called Row lines
