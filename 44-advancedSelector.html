All Selector in CSS
  • Element Selector 
                    p{
                    color:red;
                }
  • ID Selector 
                    #blackBG{
                    background-color:black;
                }
  • Class Selector 
                     .blackBG{
                    background-color:black;
                 }
  • Pseudo Class and Pseudo Element 
                    a:hover{
                    color:red;
                }
                a:first-line{
                    color:yellow;
                }
  • Attribute Selector 
                    a[href="https://google.com"]{
                    color:red;
                }
  • Universal Selector 
                    *{
                    border: 2px solid red;
                }
  • Combinator Selector
    1. Descendant Combinator(child at any stage) 
                              .red-p span{ /*.red-p class inside span */
                            color:red;
                        }
    2. Child Combinatory(Direct Child) 
                              .rep-p > span{
                            color : yellow;
                        }
    3. Adjacent Combinator(Right After Sibling) 
                              .red-p +h1{
                            color:blue;
                        }
    4. General Sibling Combinator(All siblings) 
                              .red-p ~ h1{
                            color:yellow;
                        }
  • Add Selector 
                    h1,p,.box{
                    background-color:red;
                }
