Colors in CSS

In CSS Colors can be specified using ->

  • Predefined Color name

    Modern Browser Support 140 Named Colors

  • RGB(Red-Green-Blue) 
                    
      
                    
    • rgb(red,green,blue)
      • 
                    
    • Color value between 0 and 255
      • 
                    
    • Black : rgb(0,0,0)
      • 
                    
    • White : rgb(255,255,255)
      • 
                    
    • Red : rgb(255,0,0)
      • 
                    
    • Green : rgb(0,255,0)
      • 
                    
    • Blue : rgb(0,0,255)
      •
  • RGBA(Red-Green-Blue-Alpha) 
                    
      
                    
    • rgba(red,green,blue,alpha)
      • 
                    
    • alpha parameter is a number between 0.0(fully transparent) and 1.0(not transparent at all)
                    
      • 
                    
    • Example :- rgba(255,99,71,0.5)
      •
  • HEX(ex- #FF03AA) 
                    
      
                    
    • #rrggbb
      • 
                    
    • hexademinal values between 00 and ff (same as decimal 0-255)
      • 
                    
    • Red -> #ff0000
      • 
                    
    • Black -> #000000
      • 
                    
    • White -> #ffffff
      •
  • HSL(Hue-Saturation-Lightness) 
                    
      
                    
    • hsl(hue,saturation,lightness)
      • 
                    
    • Hue is a degree on the color wheel from 0 to 360
      • 
                    
    • 0 is red
      • 
                    
    • 120 is green 
      • 
                    
    • 240 is blue 
      • 
                    
    • Saturation is a percentage value,0% means a shade of gray, and 100% is the full color
      • 
                    
    • Lightness is also a percentag,0% is black , 50% is neither light or dark, 100% is white
      • 
                    
    • RED : hsl(0,100%,50%)
      •
  • HSLA(Hue-Saturation-Lightness-Alpha) 
                    
      
                    
    • hsla(hue,saturation,lightness,alpha)
      • 
                    
    • alpha parameter is a number between 0.0(fully transparent) and 1.0(not transparent at all)
      • 
                    
    • hsla(0,100%,50%,0.5)
      •
